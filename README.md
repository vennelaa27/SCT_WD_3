# 🎮 ChronoX Tic Tac Toe Pro

A modern and responsive **Tic Tac Toe Web Application** developed as **Task 3** for the **SkillCraft Technology Web Development Internship**.

The project offers both **Player vs Player** and **Player vs Computer** modes with a clean UI, responsive design, and interactive gameplay built using HTML, CSS, and JavaScript.

---

## 🚀 Live Features

- 🎮 Player vs Player Mode
- 🤖 Player vs Computer Mode
- 🧠 Easy & Hard AI Difficulty
- 🏆 Winner Detection
- 🤝 Draw Detection
- 📊 Live Scoreboard
- 💾 Local Storage Support
- 🌙 Dark / Light Theme Toggle
- 🔄 Restart Game
- ➕ New Game
- 📱 Fully Responsive Design
- ✨ Glassmorphism UI
- 🎯 Smooth Animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Font Awesome
